## What Is ChestLink?
**Turn any chest you place into a portable virtual inventory accessible from any world.**

ChestLink allows players to bind a **real-placed chest** to a **virtual inventory** that can be opened from anywhere on the server ‚Äî any world, any dimension, any map. It works like a portable storage system, but instead of using backpacks, players can link their storage to a physical chest they have created. This creates an immersive and intuitive storage system perfect for Survival, SMP, RPG, and Multi-World servers.

---

### Features

- Bind a physical chest to a virtual, server-wide inventory
- Players start with **1** single-chest inventory by default
- Additional single or double-chest inventories unlocked via permissions
- GUI-based inventory manager (`/chestlink list`)
- Rename, reset, and delete inventories
- Read-only admin inspection mode
- Admin inventory editing with full safety validation
- Strict placement rules to prevent dupes and exploits
- If the original bound chest is broken, its virtual inventory is wiped
- Multi-world, multi-dimension support
- Easy to use, beginner-friendly, and fully configurable

---

### How It Works

1. Player holds a chest block.
2. Runs `/chestlink bind <name>`.
3. Places the chest and punches it.
4. A virtual inventory is created and linked.
5. Player can now open it anywhere using `/chestlink open`

If the original bound chest is **broken**, the inventory is **immediately deleted and wiped**.
